## Overview

![screenshot](screenshot/127.0.0.1_5500_index.html_.png)

**Device Shop Checkout** is a clean and responsive checkout page designed to display selected products, total prices, and a user information form in one elegant layout.  
The layout is built using a combination of **CSS Grid** and **Flexbox**, while the color scheme is managed through **CSS custom properties (variables)** for better theme control.

This project was created as a submission to a [devChallenges.io](https://devchallenges.io/challenge/apple-shop-checkout-page-challenge) challenge.

---

### What I learned

- Creating a complex layout using **CSS Grid** to divide the sidebar (cart) and checkout form sections.  
- Using **CSS variables** to easily manage colors and theme.  
- Building a semantic HTML structure with elements such as `header`, `main`, `aside`, and `footer`.  
- Designing clean and accessible input fields and form layout.  
- Adding hover transitions for subtle interactivity.
